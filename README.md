# ruzzle - implementation of solution for game

Inspired from [The Trie Data Structure](https://www.toptal.com/java/the-trie-a-neglected-data-structure)

## Run
```bash
$ bundle install
```

### Rubocop
```bash
$ bin/rake cop
```

### Test
```bash
$ bin/rake test
```
